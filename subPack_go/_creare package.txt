

	Nella directory che contiene il main 
		ho definito 2 cartelle una per package
		
		in ogni package le funzioni che saranno chiamate dall'esterno devono iniziare con la MAIUSCOLA 
		in ogni cartella package bisogna  eseguire go mod init nome modulo
		alla fine nella cartella main 
			oltre a go mod init mioMain.go
			devo eseguire per ogni package  (mi serve a dire che le cartelle si trovano sotto la cartella main
				go mod edit -replace uno=/uno
				go mod edit -replace due=/due
			alla fine go mod tidy 
		e quindi 
			go build mioMain.go
